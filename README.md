1. Make sure you are running you calculator api proyect first on localhost:3000

2. Change name of .env_example to .env

3. Install using npm

   `$ npm install`

4. Run with hot reload for dev

   `$ npm run dev`

5. Put the login creds, created by seed on api

6. After login go to operations and select add_credits, add a number and run.
   This should give you credits for perform the other operations.
